Oferta académica

Recuperación de datos HDD y SSD: Los estudiantes aprenderán las técnicas y herramientas necesarias para recuperar y proteger información almacenada en dispositivos electrónicos. Incluyendo diagnóstico de problemas de almacenamiento, recuperación de datos borrados o dañados, reparación de sistemas de archivos y clonación y respaldo de datos.

Temas a desarrollar:

  • Configurar y administrar un sistema operativo para el uso eficiente de los dispositivos de almacenamiento de datos
  • Reparar daños lógicos en las unidades de almacenamiento
  • Recuperar datos perdidos por falas o mal uso
  • Personalizar la seguridad mediante políticas de encriptación
